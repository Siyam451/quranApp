## ✨ Overview

Quran App is a beautifully designed Islamic app that provides essential daily features:

* 📖 Quran reading
* 🕌 Prayer time
* 📿 Tashbeeh Counter
* 🤲 Daily duas
* 🌙 Namaz guide
* 📍 Qibla Finder
* 🤲 Types of Special Namaj Section
